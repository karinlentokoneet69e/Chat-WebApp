Ohjelmistoprojekti:
Yksinkertainen lähiverkossa toimiva chat nettisivu, joka on tehty käyttäen frontendinä javascriptia, html, css ja backendinä nodea. Käyttäjien tiedot on tehty alustavasti tallentumaan json samoinkuin että onko käyttäjä chatin moderaattori vai ei. Chatissa on myös muutama "!" komento, jolla voi muunmuassa poistaa/lisätä moderaattoreita ja tyhjentää chatin viesteistä. Chatin käynnistys toimii niin, että joko avaa client.html tai avaa cmd:n kansiosta missä on server.js ja kirjoittaa tämän "npm install && node server.js".
